# Merge3D project page

Static homepage for:

**Merge3D: Efficient 3D Multimodal LLMs via Joint 2D-3D Token Merging**

## Files

- `index.html` — main page
- `style.css` — page styling
- `script.js` — small BibTeX copy helper
- `assets/` — figures converted from the paper

## Local preview

From this folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Before publishing

- replace the placeholder Paper / arXiv / Code buttons in `index.html`
- update the BibTeX entry if the final publication venue changes
- optionally replace author text with homepage links
